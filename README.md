# shell-boilerplate
A boilerplate generator for a simple shellscript.

Supported shells:
- sh
- bash

## Usage
```
$ make <shell>
```
Generate a `bash` boilerplate
```
$ make bash
$ mv boilerplate.sh myscript.sh
```
